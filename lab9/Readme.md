## Steps to run:
1) Open terminal in directory containing the Haskell file.  
2) Compile using: `ghc mergesort.hs`  
3) Run using: `./mergesort`  
4) Enter integer list elements as prompted.  

## Example Uasge
```
$ ghc mergesort.hs
$ ./mergesort
$ Enter elements of the list separated by spaces:
4 7 9 2 1 0 6
Original List:
[4,7,9,2,1,0,6]
Sorted List:
[0,1,2,4,6,7,9]
```
